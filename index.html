About

I'm a Senior Software Quality Engineering Consultant, helping teams design test strategy, build automation, and raise release quality. Before moving into quality engineering, I spent years as a software developer — so I approach testing from the inside, understanding how systems are actually built, not just how they're supposed to behave.

That dual perspective means I can work across the full delivery lifecycle: shaping test plans, driving automation, triaging bugs, and collaborating closely with engineering teams to close the gap between "it works" and "it's proven to work."

Outside of client work, I'm building a personal full-stack blog platform: an ASP.NET Core Web API backend with JWT authentication and role-based access, paired with a React + TypeScript frontend using Tiptap for rich editing and TanStack Query for data fetching.
